# Guidelines for JSOC Packages

## @jsoc/core
- Must stay framework-agnostic
---

## @jsoc/react
  - NAMING CONVENTIONS
    - Public components:
      - Always prefixed with Jsoc*
      - Example: JsocGrid

    - Internal adapters:
      - Wrapper Component Name + Adapter Component Short Name
      - Examples: JsocGridMui, JsocGridAg
---
